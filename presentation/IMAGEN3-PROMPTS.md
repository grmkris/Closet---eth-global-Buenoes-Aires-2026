# IMAGEN 3 OPTIMIZED SLIDE PROMPTS
## StyleAgent ETH Global Presentation

**Purpose:** Ultra-optimized prompts for Google Imagen 3 based on image generation and presentation design best practices.

**Design Philosophy:**
- Visual hierarchy: Eye flows naturally to key elements
- Consistency: Unified color palette and style across all slides
- Polish: Production-quality, not "AI-generated" look
- Clarity: Message supports speech, doesn't compete
- Impact: Each slide has ONE clear focal point

**Global Style Guide:**
- **Primary Colors:** Deep blue (#1E40AF), Purple (#7C3AED), White (#FFFFFF)
- **Accent Colors:** Green (#10B981) for success, Orange (#F59E0B) for in-progress
- **Typography:** Modern sans-serif, high contrast, clean hierarchy
- **Aesthetic:** Tech startup pitch deck (think Stripe, Linear, Vercel)
- **Image Quality:** Photorealistic where appropriate, clean illustration elsewhere
- **Lighting:** Professional, slightly elevated exposure, subtle gradients
